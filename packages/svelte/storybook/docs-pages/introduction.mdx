# Velkommen til STAF sitt komponentbibliotek!

Dette komponentbiblioteket bygger på [Digdirs designsystem](https://www.designsystemet.no/) som inneholder en rekke
grunnleggende komponenter som er utformet på en universell måte. I ulikhet med Digdirs komponenter er våre komponenter
tilpasset STAFs designprofil og er derfor tilpasset STAFs farger, fonter og ikoner. I tillegg, er komponenten også
skrevet i [Svelte](https://svelte.dev/), som er et rammeverk som er lettere å bruke enn React og som gir raskere lastetid.

## Komponenter
