# Velkommen til STAF sitt komponentbibliotek!

Dette komponentbiblioteket bygger på <a href="https://www.designsystemet.no/" target='\_blank' style={{textDecoration:"underline", color: "#4c76ba"}}>Digdirs designsystem</a> som inneholder en rekke
grunnleggende komponenter som er utformet på en universell måte. I ulikhet med Digdirs komponenter er våre komponenter
tilpasset STAFs designprofil og er derfor tilpasset STAFs farger, fonter og ikoner. I tillegg, er komponenten også
skrevet i <a href="https://svelte.dev/" target='\_blank' style={{textDecoration:"underline", color: "#4c76ba"}}>Svelte</a>, som er et rammeverk som er lettere å bruke enn React og som gir raskere lastetid.

## Komponenter
